## Requisitos
* ptyhon3 instalado e configurado globalmente
* pip instalado e configurado globalmente

## Instalar o virtualenv
~~~bash
$ pip install virtualenv
~~~

## Criar um ambiente virtual
~~~bash
$ virtualenv venv
~~~

## Ativar o ambiente virtual
~~~bash
$ venv\Scripts\activate
~~~
